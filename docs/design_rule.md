# デザイン再現ルール

## 参考資料
- assets/reference_design.pdf
- assets/reference_fv.png

## 全体方針
参考PDFのデザインをもとに、女性向け・講座LPらしい、やわらかく上品な縦長LPとしてHTML/CSSで再現する。

PDFや画像をそのまま貼るのではなく、配色・余白・文字配置・装飾・カードデザイン・CTAの雰囲気を参考にして、Webページとして実装する。

全体としては、白をベースに、淡い水色・ミントグリーン・ピンク・ゴールドをアクセントとして使う。強いコントラストや硬いビジネス感ではなく、「清潔感」「透明感」「やさしさ」「女性らしい上品さ」「軽やかな変化」を感じるデザインにする。

## デザインの方向性
- 女性向け
- やわらかい
- 透明感がある
- 上品
- 清潔感がある
- 夢・変化・可能性を感じる
- 淡く明るい
- 押し売り感がない
- 講座・スクール・オンラインイベントらしい信頼感がある

## 避ける方向性
- 黒やネイビーを強く使った重たいデザイン
- 原色を多用した派手なデザイン
- テック感が強すぎるサイバー系デザイン
- 男性向けビジネスLPのような硬い印象
- 情報量を詰め込みすぎた圧迫感のあるレイアウト
- Canvaテンプレート感が強い安っぽいデザイン
- PDFをそのまま画像として貼るだけの実装

## 配色
参考PDFのように、以下の色味を中心に構成する。

### ベースカラー
- 白
- ごく淡い水色
- 透明感のあるライトブルー

### メインアクセント
- ミントグリーン
- 淡いターコイズ
- やわらかいピンク

### 高級感・見出しアクセント
- くすみゴールド
- ベージュゴールド

### 文字色
- 本文：濃すぎないグレー
- 見出し：ゴールド、グレー、または淡いブルーグリーン
- 強調：ミントグリーン、ピンク、ゴールドの帯で表現

## 背景
- 全体背景は白を基本にする
- セクションによって淡い水色の背景を使う
- 必要に応じて、薄いグラデーションや光のにじみのような表現を入れる
- 背景は主張しすぎず、テキストの読みやすさを優先する
- 参考PDFのように、中央に細いLP本体があり、左右に余白がある見え方を意識する

## レイアウト
- スマホファーストで作る
- LP本体の横幅は広げすぎず、中央寄せにする
- PC表示でも、スマホLPを中央に配置したような縦長レイアウトにする
- 余白を広めに取り、各セクションに呼吸感を出す
- セクションごとに背景色・カード・帯・装飾で区切る
- 情報を横に広げるより、縦に読み進めやすくする
- CTAは複数配置するが、強く売り込みすぎない

## ファーストビュー
参考PDFの1ページ目のように、以下の要素を入れる。

- 上部に淡い光・グラデーション感のある背景
- メインビジュアルまたは抽象的な明るいビジュアル
- ロゴ風の小さな要素
- 英字または日本語の大きなメインタイトル
- サブコピー
- 日程・特典・無料訴求などの重要情報
- 丸みのある大きなCTAボタン

FVは、最初に見た瞬間に「やさしい」「きれい」「プロっぽい」「自分にもできそう」と感じる雰囲気にする。

## 見出しデザイン
- 大見出しはゴールド系を使い、上品に見せる
- 英字見出しを補助的に使ってもよい
- 日本語見出しは中央寄せを基本にする
- 余白をしっかり取り、見出しだけで世界観が伝わるようにする
- 見出しの下に小さなサブコピーを入れて、セクションの意味を補足する

## 本文デザイン
- 本文は中央寄せまたは読みやすい左寄せを使い分ける
- 1行を長くしすぎない
- 行間を広めにする
- 文字サイズは小さくしすぎない
- 読み物としての余白とリズムを重視する

## 強調表現
参考PDFの3〜4ページ目のように、強調したい言葉は以下のように表現する。

- 淡いピンクの帯
- ミントグリーンの帯
- ゴールドの帯
- 角丸のラベル
- 半透明の背景
- 短いフレーズを1行ずつ重ねる

強調は、太字だけに頼らず、淡い色の帯で視覚的に見せる。

## カードデザイン
参考PDFの1ページ目・6ページ目のスケジュールカードのように、講座内容やステップはカード形式で見せる。

カードの特徴：
- 白背景
- うすい枠線
- くすみゴールドのラベル
- 影は薄め
- 角丸
- 画像や装飾を入れる場合は淡く上品にする
- DAY1 / DAY2 / DAY3 のようなラベル表現を使ってよい

## CTAボタン
参考PDFのCTAのように、丸みのある大きなボタンにする。

CTAの特徴：
- 横長の角丸ボタン
- ミントグリーン〜ピンクのグラデーション
- 白文字
- スマホで押しやすい高さ
- ボタン周辺に余白を取る
- 「無料」「オンライン」「今すぐ受け取る」などの安心感が伝わる文言を入れる

CTAは派手に煽るより、やさしく自然に押せる印象にする。

## 装飾要素
参考PDFにあるような、以下の装飾を適度に使う。

- 丸いスタンプ風の要素
- 菱形の小さなラベル
- 淡いライン
- 薄い影
- 手書き風・イラスト風の軽い装飾
- 透明感のある光の背景
- くすみゴールドの小見出しラベル

ただし、装飾を増やしすぎず、本文の読みやすさを優先する。

## セクション構成の見せ方
以下のような流れで、視覚的に変化をつける。

1. FV
   - 明るいビジュアル
   - メインコピー
   - CTA

2. 悩み・問題提起
   - 淡い水色背景
   - チェックリスト形式
   - 読者の悩みをやさしく拾う

3. 誤解・隠れた原因
   - 白背景
   - ゴールド見出し
   - 余白を広く取り、読み物感を出す

4. 解決策・新しい考え方
   - ピンク・ミントの帯で強調
   - 短いフレーズを重ねる

5. 成功の鍵・理由
   - 中央寄せ
   - 図解風のラベルや丸要素を使う

6. セミナー内容・スケジュール
   - カード形式
   - DAY1 / DAY2 / DAY3 のような構成
   - 白いカード＋ゴールドラベル

7. CTA
   - 淡い背景
   - 大きなボタン
   - 無料・オンライン・受け取りやすさを強調

## フォント
- 基本フォントは Noto Sans JP
- 見出しには、上品に見える serif 系フォントを一部使ってもよい
- 英字見出しには serif 系を使うと参考PDFに近い雰囲気になる
- 本文は読みやすさ重視で Noto Sans JP を使う

## 画像の扱い
- 参考PDFのように、女性・光・空・ノート・講座・学びを連想させる明るい画像と相性がよい
- 画像を使う場合は、角丸・薄い影・淡い枠線で上品に配置する
- 画像がない場合でも、淡いグラデーションやカード装飾で世界観を再現する

## 実装条件
- HTMLとCSSだけで作る
- JavaScriptは使わない
- フレームワークは使わない
- GitHub Pagesで公開できる構成にする
- index.html と style.css を中心に構成する
- CSSは初心者でも編集しやすいように、セクションごとにコメントを入れる
- スマホファーストで実装する
- PC表示では中央にLP本体が表示されるようにする

## 特に重視すること
- 参考PDFのような「淡く、やさしく、上品な女性向けLP」の空気感
- 余白の美しさ
- CTAの押しやすさ
- 講座LPとしての信頼感
- 読み進めたくなる縦長のリズム
- 安っぽさを避けること
